## How to Make This Repository Public on GitHub

Since repository visibility cannot be changed through code, follow these steps:

1. **Go to Repository Settings**
   - Navigate to: https://github.com/ninoyerbas/JCRM-/settings

2. **Scroll to the "Danger Zone" section** (at the bottom)

3. **Click "Change visibility"**

4. **Select "Make public"**

5. **Confirm by typing the repository name**

6. **Click "I understand, make this repository public"**
